CIMO Framework

Compliance Issue Management Oversight

Comprehensive framework for oversight of compliance remediation activities

⬇

8 Core CIMO Components

1. IM Program Adherence
  • Tracking:
    • Program SLAs
    • Issue Status Updates
2. Reporting
  • Weekly reporting to owners and approvers
  • Monthly and quarterly targeted reporting to leadership
3. Change Governance
  • Risk acceptance
  • Extensions
  • Issue/action plan change management
4. Escalation Paths
  • Reporting to Compliance Governance Forums
  • Escalation to Enterprise IM Forum when required
5. Training
  • Role-specific training
  • CIMO Framework-specific training
6. In-Line QA
  • Self-identified issue intake tollgate
  • Action plan development review
  • Dependency validation & attestation
  • Pre-validation challenge of evidence
7. Quality Control
  • Recurring sampling of:
    • Issues
    • Root Cause Analysis
    • Action Plans
    • Closures
8. Remediation Tracking
  • 30/60/90 outreach to proactively address upcoming due dates

All 8 Core components apply to all compliance issues. Application is different based on track assignment.

✓ Follow Block's Enterprise Issue Management Program
✓ Receive CIMO Standard Governance (8 core components to varying degrees)

The track determines any level of added oversight an issue receives based on risk
